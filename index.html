Event Details

The Webz Hackathon is a 24-hour event where you can work with your team to create a project from scratch. You can use any programming language you want, and you can use any tools you want. You can even use other people's code as long as you give them credit. The only rule is that you have to create something new.

At the end of the event, you will have 5 minutes to present your project to the judges. The judges will then decide which team wins the hackathon.

  • Date: January 1-2, 2022
  • Location: Online
  • Prizes: $500 for first place, $250 for second place
